# zed
My keymap for [zed](https://zed.dev). 

It's an editor I like to use. Has some weird shortcuts I've learned from visual studio, jetbrains, or just made up. 

Some ones I can remember off of the top of my head:

`Ctrl+shift+L`: delete line

`Ctrl+w`: close tab

`Ctrl+.+/`: close all tabs

`Ctrl+shift+N`: open by file name
